HTML, Hyper Text Markup Language. Is the #1 language used by web developers. A tag is anything between a less than and greater than sign.

<a> would be a tag.

An Element is a object consisting of 3 essential components. A starting tag, content / inner HTML, and an end tag. End tags are identical to Starting tags except there is a forward slash after the less than sign. Not every element requires an end tag, but placing them does not affect the code execution.

Element:
<p>This element displays text.</p>
Shorthand Ending special elements:
<img src="image.png" alt="image"/>
